🧊 基礎魔術方塊解法（互動視覺版）

1️⃣ 小花階段

目標：將白色邊塊集中在黃色中心周圍，形成 "小花"。

🌼 中心是黃色，四周白色像一朵花。

2️⃣ 白色底十字

目標：將白色邊塊對準顏色後打到底部，形成底層十字。

⬇️ 白邊塊到底部，側邊顏色要對。

3️⃣ 第一層角塊（四步法）

目標：放好白色角塊，完成整個白色底面。

✅ 使用四步法將角塊放對。

4️⃣ 第二層邊塊

目標：處理中間層的四個邊塊，對色插入。

↔️ 根據對齊方向使用左/右四步法插入。

5️⃣ 小魚公式

目標：讓頂面變成黃色十字。

➕ 使用小魚公式（前鉤公式）將「點 → L → 線 → 十字」。

6️⃣ 雙蚯蚓

目標：角塊位置輪換 → 使用右左輪轉法。

🔄 右蚯蚓翻轉 → 左蚯蚓翻轉 → 各自回位

7️⃣ 最後邊塊歸位

目標：頂層四面顏色成環，還原完成。

🎉 若三塊連線 → 後方 → 小魚公式重複 → 解開魔方！
